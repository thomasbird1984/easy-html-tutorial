Basics

I always like to use the anology of a car, likewise I like to explain a website as a car. If you think about a car it has three different major parts that become the car. For instance you have the frame, styling and then the motor. A website has something similar, html is the frame of your website it gives you the core frame of your websites structure. CSS is the styling of the car, in website terms that means that it will control everything you see visually from font colors, to the type of layout and much more. The engine of a car is like the scripting that most websites employ, scripting does things like dropdown menus, information intake from users, storage of data and file uploads are just a couple examples.

Back 10 yrs ago html used to be very basic with only a limited number of tags supported. Tags are the way you tell the browser what any given piece of content is. No a days though html has a multitude of tags that are supported dependent on the browser. Web browsers like chrome, firefox or safari all implement html tags and styles differently in addition to also lacking support for some features. This can be a headache for a developer because you have to know what tags are supported and can even be used. There are great resources though for finding out what you can use such as "caniuse.com"

HTML

Html tags are the building blocks of the web, they're typically formatted like so: <tag_name>Content to be contained</tag_name>. Html tags almost always have a starting and ending tag, in rare cases with tags like <img /> there are no closing tags instead the end of the tag contains a / signifying the end of that tag.

Many times html can be used to identify the kind of content its holding, for instance <p> contain paragraph content and <a> anchor tags contain links to other places in a page or another website. Typically html tags always have starting and ending tags at the beginning and end of whatever they contain. Tags should enclose the content they are describing, in rare cases there are tags that don't have closing tags.
